# File-Sharing-App-Django

This is a simple File-sharing Web -app which has been created using Django Framework.

Functionalities:
- Login
- Logout
- Register
- Upload file(.png, .pdf, .jpg, etc)
- Delete File
- Download File
- Multiple Users

Tools Used:
- Django 3
- Python >=3 version
- Html
- CSS
- Bootstrap
- Jquery
- Pillow
